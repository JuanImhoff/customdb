# customdb

A lightweight key–value storage engine written in **C++17**, with a strong focus on
clean architecture, correctness, and portability.

This project is built as a learning and portfolio exercise to demonstrate:
- modern C++ practices
- CMake-based builds
- automated testing
- professional version control workflows

---

## Requirements

### Build tools
- CMake ≥ 3.16
- A C++17-compatible compiler
  - GCC (recommended, tested)
  - Clang (tested)
  - MSVC (should work, not primary target)

### Tested environments
- Linux (Ubuntu, incl. WSL)
- GitHub Actions (Ubuntu)
